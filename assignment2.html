WRITE WITH COPIOUS EXAMPLES,EXPLAIN THE SIMILARITIES AND DIFFERENCES BETWEEN CONST,VAR,LET IN JAVASCRIPT

The similarities between CONST,VAR,LET in JavaScript are

  1. These three keyword in JavaScript are used for declearing variables
  2. They are all used to store values
  3. They can all be used within a function,loop,and other block of code

The diffrences between CONST,VAR,LET in JavaScript are

  1. VAR was the first keyword in declearing variables ,one of the maib diffrences between VAR and other keywords is that it can be redecleared (variable mutation).
  2. LET is one of the new keywords created what makes it diffrent from others is that it cannot be redecleared but it can be reassigned
  3. CONST is use to store valued that cannot be changed

Examples of the diffrences include

(1) VAR EXAMPLE

<.script.>
Var myName="iman";
var myName="ola"
myName="Ola";
alert (myName);
<./script>

NOTE: This shows in this example that VAR can be redeclearedand be reassigned.

(2) LET EXAMPLE

<.script.>
LET myName="iman";
myName="Ola";
alert (myName);
<./script>

NOTE: This shows in this example that LET can only be reassigned and cannot be redecleared.

(3) CONST EXAMPLE <.script>
const retireAge=65;
const currentAge=prompt('how old are you');
const yearLeft=retireAge-currentAge;
alert(`you have ${yearLeft} until retirement`);
<./script>

NOTE:In this example it indicate that CONST stores a value and it cannot be changed.
